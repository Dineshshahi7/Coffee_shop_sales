# Coffee Shop Sales Analysis Dashboard

## Project Overview
This project focuses on analyzing coffee shop sales data to gain meaningful business insights and present them through an interactive dashboard. The goal of this project is to simulate a real-world retail analytics scenario and support data-driven decision-making using sales data.

The analysis helps identify sales trends, best-selling products, peak business hours, and store-wise performance.

## Dataset Information
- **Dataset Name:** Coffee Shop Sales Dataset  
- **Source:** Public practice dataset inspired by real-world coffee shop operations (commonly shared by Maven Analytics and Kaggle)  
- **Type:** Synthetic but realistic retail sales data  
- **Format:** Excel (.xlsx)

### Key Columns:
- Transaction Date  
- Transaction Time  
- Store Location  
- Product Category  
- Product Name  
- Quantity Sold  
- Unit Price  
- Total Sales  

## Objectives
- Analyze overall sales performance
- Identify top-selling products and categories
- Understand peak sales hours and days
- Compare store-wise revenue performance
- Create a clear and visually appealing dashboard for business insights

## Dashboard Insights
- Daily and monthly sales trends  
- Peak sales hours analysis  
- Best-performing products and categories  
- Store-wise sales comparison  
- Revenue contribution by product type
